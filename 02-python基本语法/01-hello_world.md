python 环境部署成功后，就可以开始编写第一句 python 代码了。

新建 hello_world.py 文件，并使用ST打开，输入以下代码

```python
print('hello world!')
```

使用`CTRL + S` 快捷键进行保存，也可以通过工具栏选择`File -> Save` 进行文件保存。

使用`F5` 快捷键运行此 python 脚本，运行成功后会在新标签页`*REPL* [python]`显示以下输出

```python
hello world!

***Repl Closed***
```
